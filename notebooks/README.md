# Python Notebook

This folder contains a Python notebook version of the QILT graduate employment analysis.

Notebook:

```text
qilt_employment_analysis.ipynb
```

The notebook covers:

- loading cleaned QILT CSV outputs;
- CIS KPI calculation;
- salary ranking by study area;
- employment and salary scatter plot;
- gender pay gap comparison;
- university-level outcome review;
- correlation analysis;
- summary insights.

The main project pipeline is still managed by R scripts, but this notebook shows the same analysis thinking in Python for data analyst portfolio purposes.
