# Resume Project Description

## English Resume Version

**Australian Graduate Employment Outcomes Analysis | R, Python, SQL, SQLite, QILT GOS-L**

- Extracted and cleaned public QILT GOS-L graduate outcomes data from a multi-sheet Excel workbook using Python and R.
- Analysed employment rates, median salaries, salary growth, and gender pay gaps across Australian study areas and universities.
- Identified that Computing and Information Systems graduates achieved 91.0% medium-term full-time employment and AUD 100,000 medium-term median salary.
- Built reusable SQL queries and a SQLite database workflow for KPI analysis, ranking, benchmarking, and gender pay gap reporting.
- Created a portfolio dashboard preview and generated a reproducible HTML report with visual insights for stakeholder-style communication.

## LinkedIn / GitHub Project Summary

I built an end-to-end data analysis portfolio project using real QILT GOS-L public graduate outcomes data. The project focuses on Computing and Information Systems graduates in Australia and compares employment, salary, salary growth, and gender pay gap outcomes across study areas and universities.

The workflow includes Python-based Excel extraction, R-based cleaning and analysis, SQL queries, SQLite database loading, visualisation, exploratory modelling, data quality checks, and an HTML report.

## Interview Short Pitch

This project analyses real QILT graduate outcomes data to understand how Computing and Information Systems graduates perform in employment and salary outcomes. I extracted data from a public multi-sheet Excel workbook, cleaned and transformed it, created KPIs and charts, added SQL and SQLite workflows, and generated a final HTML report. The key finding is that CIS graduates had a 91.0% medium-term full-time employment rate and AUD 100,000 median salary, while still showing an 8.7% gender pay gap.
